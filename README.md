# Pensieve-deployment
deploy the pensieve 

我们搭建了pensieve的运行环境，并根据原作提供的数据进行训练，本库记载了项目搭建的流程以及实验中涉及到的问题，模型以及结果。

最初时训练的模型在目录pre中，熵值的权重为项目最初预设的0.5，训练了将近两周的时间，但是没有取得预期效果。所以采用作者建议的逐渐降低熵值权重的训练方法。

逐渐降低熵值权重的训练策略如下：

   熵值权重   迭代次数
   5         21100
   4         40000
   3         60000+
   2         26900
   1         48600
   0.5       12000
   0.4       13000
   0.3       10700
   0.2       10200
   0.1       160000+


   model-cache为部分训练过程中生成的模型
   model-point为节点模型
